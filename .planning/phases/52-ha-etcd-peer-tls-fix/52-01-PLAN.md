// Verdict is the IPAM probe outcome consumed by lifecycle and the doctor check.
type Verdict string

const (
    VerdictIPPinned   Verdict = "ip-pinned"
    VerdictCertRegen  Verdict = "cert-regen"
    VerdictUnsupported Verdict = "unsupported" // nerdctl & "command not found"
)

// ProbeIPAM runs the full lifecycle simulation against the given runtime CLI
// (docker / podman / nerdctl) and returns the verdict + a structured reason.
// Cleanup is deferred and runs even on failure.
//
// Behavior contract:
//   - nerdctl: short-circuit to VerdictUnsupported BEFORE any container ops
//     (RESEARCH PIT-4 and Open Question on probe image avoidance).
//   - docker/podman: full 8-step lifecycle (RESEARCH "Probe: Concrete Definition").
//   - On any non-"command not found" runtime error: return VerdictCertRegen
//     with the wrapped error so callers can warn-and-proceed.
type Verdict string
func ProbeIPAM(binaryName string) (Verdict, string, error)

| inspected IP → ProbeIPAM call site | string from `docker inspect` is passed back into `--ip` flag |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-52-01-01 | Tampering | ipamprobe.go IP capture | mitigate | Validate captured IP with `net.ParseIP(ip)` before passing to `--ip`; reject empty or malformed strings (V5 input validation) |
| T-52-01-02 | DoS (resource exhaustion) | scratch network/container leak on probe failure | mitigate | Deferred cleanup runs even on early failure; cleanup is best-effort `_ =` so cleanup of one resource cannot block the other; unique timestamped names prevent stale-resource collisions |
| T-52-01-03 | Information Disclosure | probe network on shared subnet | accept | Probe uses dedicated 172.200.0.0/24 subnet documented as non-overlapping; no sensitive data ever traverses the probe net (only `sleep 30`) |
| T-52-01-04 | DoS | probe runtime hang | mitigate | `sleep 30` bound + each runtime CLI call returns within seconds; total probe upper-bound matches CONTEXT.md "10–30s acceptable" |
</threat_model>
