    Follow the `localPathCVECheck` pattern with injected dependencies:

    ```go
    // hostMountPathCheck verifies that configured host mount paths exist
    // on the host filesystem. Skips gracefully when no mount paths are provided.
    type hostMountPathCheck struct {
        // getMountPaths returns host paths to check. Returns nil when no
        // mount configuration is available (e.g., kinder doctor runs standalone).
        getMountPaths func() []string
        // statPath checks if a path exists. Injected for testing.
        statPath func(string) (os.FileInfo, error)
    }

    func newHostMountPathCheck() Check {
        return &hostMountPathCheck{
            getMountPaths: func() []string { return nil }, // no-op: skip when no config
            statPath:      os.Stat,
        }
    }

    func (c *hostMountPathCheck) Name() string       { return "host-mount-path" }
    func (c *hostMountPathCheck) Category() string    { return "Mounts" }
    func (c *hostMountPathCheck) Platforms() []string { return nil } // all platforms
    ```

    Run() logic:
    1. Call `c.getMountPaths()` to get paths.
    2. If nil or empty slice, return a single `Result` with Status `"skip"`, Message `"no mount paths configured"`.
    3. For each path, call `c.statPath(path)`:
       - If `os.IsNotExist(err)`: append a Result with Status `"fail"`, Message identifying the missing path, Reason `"kinder create cluster will fail when this path is mounted into a node"`, Fix `fmt.Sprintf("mkdir -p %q", path)`.
       - If `err != nil` (other error): append a Result with Status `"warn"`, Message about inaccessible path with the error.
       - If no error: append a Result with Status `"ok"`, Message `fmt.Sprintf("host path %q exists", path)`.
    4. Return all results.

    **Check 2: dockerDesktopFileSharingCheck**

    Follow the `daemonJSONCheck` pattern with injected readFile and homeDir:

    ```go
    // dockerDesktopFileSharingCheck verifies that configured host mount paths
    // are covered by Docker Desktop's file-sharing configuration on macOS.
    type dockerDesktopFileSharingCheck struct {
        readFile      func(string) ([]byte, error)
        homeDir       func() (string, error)
        getMountPaths func() []string
    }

    func newDockerDesktopFileSharingCheck() Check {
        return &dockerDesktopFileSharingCheck{
            readFile:      os.ReadFile,
            homeDir:       os.UserHomeDir,
            getMountPaths: func() []string { return nil }, // no-op: skip when no config
        }
    }

    func (c *dockerDesktopFileSharingCheck) Name() string       { return "docker-desktop-file-sharing" }
    func (c *dockerDesktopFileSharingCheck) Category() string    { return "Mounts" }
    func (c *dockerDesktopFileSharingCheck) Platforms() []string { return []string{"darwin"} }
    ```

    Run() logic:
    1. Call `c.getMountPaths()`. If nil or empty, return `skip` result with `"no mount paths configured"`.
    2. Get home dir via `c.homeDir()`. If error, return `warn` result with `"cannot determine home directory"`.
    3. Construct settings path: `filepath.Join(home, "Library", "Group Containers", "group.com.docker", "settings-store.json")`.
    4. Read the file via `c.readFile(settingsPath)`. If error, return `warn` result with message: `"cannot read Docker Desktop settings — file sharing check skipped"`. Reason: `"Docker Desktop may not be installed or settings file moved"`.
    5. Parse JSON into a `map[string]interface{}`. Extract `"FilesharingDirectories"` key. If missing, use default dirs: `[]string{"/Users", "/Volumes", "/private", "/tmp", "/var/folders"}`.
    6. For each mount path from `getMountPaths()`, check if it's covered by any sharing directory using the `isPathCovered` helper.
    7. If covered: append `ok` result.
    8. If NOT covered: append `warn` result with Message identifying the uncovered path, Reason `"Docker Desktop will not be able to mount this path into containers"`, Fix `"Open Docker Desktop → Settings → Resources → File Sharing and add this path"`.
    9. Return all results.

    **Helper function:**

    ```go
    // isPathCovered checks whether path falls under any of the shared directories.
    // Uses directory-terminated prefix matching to avoid false positives
    // (e.g., /Userspace/foo must NOT match /Users).
    func isPathCovered(path string, sharedDirs []string) bool {
        for _, dir := range sharedDirs {
            if path == dir || strings.HasPrefix(path, dir+"/") {
                return true
            }
        }
        return false
    }
    ```

    ```go
                }
            }
            if len(dirs) > 0 {

            }
        }
    }
    ```
