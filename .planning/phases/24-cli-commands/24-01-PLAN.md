1. Create `pkg/cmd/kind/doctor/doctor.go` with package `doctor`. Follow the Cobra leaf command pattern:

   - `NewCommand(logger log.Logger, streams cmd.IOStreams) *cobra.Command` — returns a cobra.Command with:
     - `Args: cobra.NoArgs`
     - `Use: "doctor"`
     - `Short: "Checks prerequisite binaries and reports actionable fix messages"`
     - `Long: "Checks for required binaries (container runtime, kubectl) and exits 0 if all ok, 1 on failure, 2 on warnings"`
     - `RunE` calls `runE(streams)`

   - Define an unexported `result` struct: `name string`, `status string` (one of "ok", "warn", "fail"), `message string`.

   - `runE(streams cmd.IOStreams) error`:
     - Collect all check results into `[]result` slice FIRST, then print and exit. Do NOT use `defer` for anything that must run before exit (os.Exit bypasses defers).

     - **Container runtime check:** Iterate over docker, podman, nerdctl. For each, call `checkBinary(name)`. If the binary is found on PATH AND responds to `version` (or `-v` fallback), mark as "ok". Stop at first "ok" runtime found. If NO runtime found at all, append a "fail" result for "container-runtime" with message: `"no container runtime found — install Docker (https://docs.docker.com/get-docker/), Podman (https://podman.io/getting-started/installation), or nerdctl (https://github.com/containerd/nerdctl)"`. If a runtime is found on PATH but `version` command fails (e.g., daemon not running), append a "warn" result with message: `"<name> found but not responding — is the daemon running?"`.

     - **kubectl check:** Call `checkBinary("kubectl")`. If not found on PATH, append "fail" result with message: `"kubectl not found — install from https://kubernetes.io/docs/tasks/tools/"`. If found but `kubectl version --client` fails, append "warn" result. If found and working, append "ok" result.

     - **Print results:** For each result, write to `streams.ErrOut` (NOT stdout):
       - `"[ OK ] <name>\n"` for ok
       - `"[WARN] <name>: <message>\n"` for warn
       - `"[FAIL] <name>: <message>\n"` for fail

     - **Exit logic:** After printing all results:
       - If any "fail" results: `os.Exit(1)`
       - Else if any "warn" results: `os.Exit(2)`
       - Else: `return nil` (Cobra exits 0)

   - `checkBinary(name string) (found bool, working bool)`:
     - Use `osexec.LookPath(name)` — if error, return `false, false`.
     - If found, run `exec.Command(name, "version")` via `exec.OutputLines`. If error or empty output, try `exec.Command(name, "-v")` via `exec.OutputLines`. If both fail, return `true, false` (found but not working). If either succeeds, return `true, true`.
     - Import `osexec "os/exec"` for LookPath and `"sigs.k8s.io/kind/pkg/exec"` for OutputLines.

   - For kubectl specifically, use `exec.Command("kubectl", "version", "--client")` instead of bare `version` to avoid contacting the API server.

   - Include Apache 2.0 copyright header.
   - Imports: `"fmt"`, `"os"`, `osexec "os/exec"`, `"github.com/spf13/cobra"`, `"sigs.k8s.io/kind/pkg/cmd"`, `"sigs.k8s.io/kind/pkg/exec"`, `"sigs.k8s.io/kind/pkg/log"`.
