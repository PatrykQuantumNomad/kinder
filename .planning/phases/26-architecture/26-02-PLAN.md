   to:
   `func tryUntil(ctx context.Context, until time.Time, try func() bool) bool`

3. Replace the `time.Sleep(500 * time.Millisecond)` inside tryUntil with a select that checks context cancellation:
   ```go
   select {
   case <-ctx.Done():
       return false
   case <-time.After(500 * time.Millisecond):
   }
   ```

4. Update `waitForReady` signature from:
   `func waitForReady(node nodes.Node, until time.Time, selectorLabel string) bool`
   to:
   `func waitForReady(ctx context.Context, node nodes.Node, until time.Time, selectorLabel string) bool`

5. Update the tryUntil call inside waitForReady from:
   `return tryUntil(until, func() bool {`
   to:
   `return tryUntil(ctx, until, func() bool {`

6. Update the call site in Execute() (line ~87) from:
   `isReady := waitForReady(node, startTime.Add(a.waitTime), selectorLabel)`
   to:
   `isReady := waitForReady(ctx.Context, node, startTime.Add(a.waitTime), selectorLabel)`

7. The `node.Command(` inside the waitForReady closure (line ~104) should also be updated to `node.CommandContext(ctx,` for consistency. Note: inside the closure, `ctx` refers to the `context.Context` parameter of `waitForReady`, not the `*ActionContext`.

1. Add `"context"` to the import block.

   - `tryUntil(time.Now().Add(5*time.Second), ...)` becomes `tryUntil(context.Background(), time.Now().Add(5*time.Second), ...)`
   - Same for all other test calls.

3. Add a NEW test for context cancellation:
```go
func TestTryUntil_RespectsContextCancellation(t *testing.T) {
    ctx, cancel := context.WithCancel(context.Background())
    cancel() // cancel immediately
    result := tryUntil(ctx, time.Now().Add(5*time.Second), func() bool {
        return false
    })
    if result {
        t.Error("tryUntil should return false when context is already cancelled")
    }
}
```
