```go
package create

// addonResult captures the outcome of running a single addon action.
type addonResult struct {
    name    string
    enabled bool
    err     error

func logAddonSummary(logger log.Logger, results []addonResult)

// logMetalLBPlatformWarning prints a warning on macOS/Windows that MetalLB

func logMetalLBPlatformWarning(logger log.Logger)
```

From pkg/log/types.go:
```go
type Logger interface {
    Warn(message string)
    Warnf(format string, args ...interface{})
    Error(message string)
    Errorf(format string, args ...interface{})
    V(Level) InfoLogger
}

type InfoLogger interface {
    Info(message string)
    Infof(format string, args ...interface{})
    Enabled() bool
}

type Level int32
```

Test logger must implement both Logger and InfoLogger interfaces.
The V(Level) method should return an InfoLogger -- the testLogger can return itself.
