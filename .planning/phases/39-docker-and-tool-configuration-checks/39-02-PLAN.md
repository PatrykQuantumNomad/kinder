   - Define `kubectlVersionSkewCheck` struct with `lookPath func(string) (string, error)` and `execCmd func(name string, args ...string) exec.Cmd` fields

   - Run() logic:
     a. If lookPath("kubectl") fails, return skip "(kubectl not found)" -- the existing kubectlCheck handles the "kubectl missing" case

     f. Compare: `clientMinor := clientVer.Minor()`. Compute `diff` as the absolute difference between clientMinor and referenceK8sMinor (use a simple int subtraction with abs, since both are uint -- cast to int first)
     g. If diff > 1, return warn with message like "kubectl v1.28.4 (client minor 28, reference 31)" and reason "kubectl version skew exceeds +/-1 minor version from Kubernetes 1.{referenceK8sMinor}" and fix "Update kubectl: https://kubernetes.io/docs/tasks/tools/"
     h. If diff <= 1, return ok "kubectl version compatible (v{gitVersion})"
   - Import `sigs.k8s.io/kind/pkg/internal/version` for ParseSemantic. Use `osexec` alias for `os/exec` (same as tools.go).
