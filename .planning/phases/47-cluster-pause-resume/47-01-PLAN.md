    Create new package `pkg/cluster/internal/lifecycle` (Go package comment in doc.go: "Package lifecycle implements cluster pause/resume orchestration and shared container-state helpers.").

    In `state.go` export the following helpers:

    1. `func ContainerState(binaryName, containerName string) (string, error)` — runs `exec.Command(binaryName, "inspect", "--format", "{{.State.Status}}", containerName)`, returns the trimmed first output line (e.g., "running", "exited"). Wrap errors with `pkg/errors`.

    2. `func ClusterStatus(binaryName string, nodes []nodes.Node) string` — iterates nodes, calls ContainerState for each, returns:
       - "Running" if all states == "running"
       - "Paused" if all states == "exited" (treat "created", "dead" as stopped too)
       - "Error" otherwise (mixed, empty, or any inspect error)

    3. `func ResolveClusterName(args []string, provider *cluster.Provider) (string, error)` — implements no-arg auto-detect rule per CONTEXT.md D-04 (Command UX & output → No-arg behavior):
       - len(args)==1 → return args[0]
       - len(args)==0 → call provider.List(); switch on count: 0=error "no kind clusters found", 1=return that name, >1=error "multiple clusters found; specify one: a, b, c".

    4. `func ClassifyNodes(allNodes []nodes.Node) (cp, workers []nodes.Node, lb nodes.Node, err error)` — splits per node Role():
       - control-plane → cp slice (sorted via nodeutils.ControlPlaneNodes for determinism)
       - worker → workers slice
       - external-load-balancer → lb (single; nil if absent)
       Returns wrapped error if Role() fails on any node.

    5. `func ProviderBinaryName() string` — auto-detect docker/podman/nerdctl in PATH (copy logic from pkg/internal/doctor/clusterskew.go realListNodes; returns "" if none).

    For testability, ContainerState and ClusterStatus must accept a `cmder` injection point so state_test.go can substitute a fake. Pattern: define `type Cmder func(name string, args ...string) exec.Cmd` package var (default = exec.Command) and have ContainerState read it. Tests swap and restore via t.Cleanup.

    No new external dependencies. All imports already in go.mod.
