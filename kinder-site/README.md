# kinder-site

Documentation site for [kinder](https://github.com/PatrykQuantumNomad/kinder) — built with [Astro](https://astro.build) + [Starlight](https://starlight.astro.build).

Live at **[kinder.patrykgolabek.dev](https://kinder.patrykgolabek.dev)**

## Commands

| Command | Action |
|:--------|:-------|
| `npm install` | Install dependencies |
| `npm run dev` | Start dev server at `localhost:4321` |
| `npm run build` | Build production site to `./dist/` |
| `npm run preview` | Preview build locally |
